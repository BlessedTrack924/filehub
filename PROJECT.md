# FileHub

## Description
FileHub is a personal app for securely sharing files between devices that I own or administer.

## Planned Features

### Security
- Single password
- Optional Face ID
- Auto-lock

### Home
- Device status
- Storage
- Quick actions

### Files
- Browse shared folders
- Upload
- Download
- Search

### Photos
- View photos the app has permission to access
- Select photos to share
- Download shared photos

### Devices
- List approved devices
- Send connection requests
- Show online/offline status

### Settings
- Change password
- Theme
- Shared folders
- Trusted devices

## Future Ideas
- Folder sync
- Transfer history
- File previews
- Favorites
