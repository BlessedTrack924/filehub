- Face ID (optional)
- Auto lock
- Change password

- Welcome screen
- Storage information
- Connected devices
- Recent transfers

- Browse shared folders
- Search files
- Upload
- Download
- Rename

- Albums
- Search
- Select multiple
- Share

## Devices
- Device name
- Online
- Offline
- Request connection
- Remove trusted device

## Settings
- Dark mode
- Notifications
- Transfer history
- Shared folders
